1. Creating a Town

To create a town, you only need enough money (usually $250–$500 depending on server config).

Use:

/town new <name>

This creates a town centered on your current chunk.

2. Claiming Land

Towny uses a grid system (chunks = 16×16 blocks). To claim more land near your town:

/town claim

Or to claim multiple chunks at once:

/town claim auto (amount of chunks)

Claiming costs money from the **town bank**.

Add money using:

/town deposit <amount>

3. Managing Residents

Invite members to your town:

/town invite <player>

Remove members:

/town kick <player>

Give ranks (assistant, helper, etc):

/town rank add <player> <rank>

4. Town Taxes & Upkeep

Each town pays a daily upkeep based on the number of claimed chunks.

Set resident taxes:

/town set tax <amount>

Set plot-specific taxes:

/town set plottax <amount>

5. Town Permissions

Control who can build, break, switch, use items, etc.

Global permissions:

/town set perm <who> <type> <on/off>

Examples:

  • /town set perm outsider build off — outsiders can’t build
  • /town set perm resident build on — let residents build in your town

6. Plot Management

Plots are chunks (16x16) inside your town. To put a plot up for sale:

/plot fs (amount)

To claim for sale plots, stand within the plot and type:

/plot claim

Unclaim a plot:

/plot unclaim

Set plot type (shop, farm, embassy, etc):

/plot set type <type>

Similarily to Section 5, you can edit perms for specific plots you own by standing inside and running:

/plot set perm <who> <type> <on/off>

7. Creating or Joining a Nation

Once you have a stable town, join a nation for protection, allies, and diplomacy. View the Nation guide under "Guides" to see how to manage them, including wars.

Create a nation:

/nation new <name>

Join a nation:

/nation join <name>

8. Useful Towny Commands

/town — main town menu
/resident — see your info
/plot ? — info about your current chunk
/town spawn — teleport to town spawn
/town set spawn — set spawn point
