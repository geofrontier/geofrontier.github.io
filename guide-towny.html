Towny Guide

Everything you need to start a town, claim land, expand, and manage residents.

1. Creating a Town

To create a town, you only need enough money (usually $250–$500 depending on server config).

Use:

/town new <name>

This creates a town centered on your current chunk.

2. Claiming Land

Towny uses a grid system (chunks = 16×16 blocks). To claim more land near your town:

/town claim

Or to claim multiple chunks at once:

/town claim outpost (if outposts are enabled)

Claiming costs money from the **town bank**.

Add money using:

/town deposit <amount>

3. Managing Residents

Invite members to your town:

/town add <player>

Remove members:

/town kick <player>

Give ranks (assistant, helper, etc):

/town rank add <player> <rank>

4. Town Taxes & Upkeep

Each town pays a daily upkeep based on the number of claimed chunks.

Set resident taxes:

/town set tax <amount>

Set plot-specific taxes:

/town set plottax <amount>

5. Town Permissions

Control who can build, break, switch, use items, etc.

Global permissions:

/town set perm <type> <on/off>

Examples:

  • /town set perm outsider off — outsiders can’t edit
  • /plot set perm resident on — let residents build in your plot

6. Plot Management

Plots are 16×16 inside your town. Stand on a plot and use:

/plot claim

Unclaim a plot:

/plot unclaim

Set plot type (shop, farm, embassy, etc):

/plot set type <type>

7. Creating or Joining a Nation

Once you have a stable town, join a nation for protection, allies, and diplomacy.

Create a nation:

/nation new <name>

Join a nation:

/nation join <name>

8. Useful Towny Commands

/town — main town menu
/resident — see your info
/map — shows claim map
/town here — info about your current chunk
/town spawn — teleport to town spawn
/town set spawn — set spawn point
