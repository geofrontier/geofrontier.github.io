Towny Quick-Start

What you actually need to know to start and run a town

Essentials Quick Start Guide

Create a Town

  • /town new [name] — $200
  • Stand in the chunk you want as your spawn
  • Must be 4 chunks away from other towns
  • /town set spawn — set your return point

Claim Land

  • /town claim — claim current chunk
  • /town claim auto 5 — claim 5 chunks outward
  • /town unclaim — remove claim
  • Money comes from town bank

Manage People

  • /town invite [player]
  • /town kick [player]
  • /town rank add [player] assistant
  • /town outlaw add [player] — ban from town

Town Money

  • /town deposit [amount] — add money
  • /town withdraw [amount] — take money
  • /town set tax [amount] — daily resident tax
  • /town — check balance and upkeep

Your First 15 Minutes

  1. Find unclaimed land — Walk until chat says "Wilderness" or use /towny map
  2. Create town — /town new YourName (costs $200, check with /balance)
  3. Set your spawn — Stand somewhere safe, type /town set spawn
  4. Claim your first chunks — /town claim auto 3 to claim 3 chunks around you
  5. Invite friends — /town invite PlayerName (they type /town accept)
  6. Set basic permissions — /town set perm outsider build off (stops randoms building)
  7. Deposit money — /town deposit 500 to cover first few days of upkeep
Done. You have a functioning town. Everything else is optional optimization.

1. Creating a Town

Requirements: $200, unclaimed land at least 4 chunks from another town's homeblock.

/town new YourTown Creates town where you stand
/town set spawn Sets return point
/town set board [msg] Sets town message

What happens next: You're now mayor. Your town has 8 claim blocks and can invite residents. That's it.

2. Claiming Land

How it works: Each chunk (16×16 blocks) costs money. First chunk $5, then +2% per chunk up to $175 max.

/town claim Claim current chunk
/town claim auto [num] Claim outward automatically
/town unclaim Unclaim (15% refund)
/town deposit [amount] Add money for claims

Claim limits: Each resident = 10 claim blocks. That's the only limit. No outposts (use embassies instead).

Tip: Claim in a circle around your homeblock, not a line. Makes your town harder to bypass.

3. Permissions (Who can do what)

Groups: resident (town members), nation, ally, outsider

Actions: build, destroy, switch (buttons/doors), itemuse, pvp

/town set perm outsider build off Most common setting
/town set perm resident pvp off No friendly fire
/town set perm ally switch on Allies can use doors

Outlaws (problem players)

/town outlaw add [player] — They can't enter your town, can be killed anywhere inside, and get jailed 2h if killed here.

4. Taxes & Upkeep (The bills)

Daily cost: $2 per claimed chunk. Paid from town bank. If bank hits $0, town enters bankruptcy and is deleted after 7 days.

Income: Set resident tax (/town set tax 50) or plot tax. Players pay automatically each day.

5. Plot Types (Special zones)

/plot set type [type] — Changes the current chunk's purpose:

  • shop — Commercial (blue C on map)
  • arena — PvP always on
  • embassy — $5k, allows foreign teleport (requires staff approval)
  • jail — Prison cells
  • wilds — Harvest only, no building

6. Nations (Join or start one)

Joining: /nation join [name] — Request to join existing nation.

Creating: Need 5 residents, $10k in bank. /nation new [name]

Benefits: Nation chat (/nc), share land borders, mutual defense.

7. Teleportation (How to move)

  • Your town: /town spawn — Free, 30s cooldown
  • Nation capital: /nation spawn — $5, 30s cooldown
  • Other towns: ❌ Not allowed directly — must use embassies
  • Your embassy: /town outpost — $500, 1h cooldown

Jail System

  • Setup: $1,000
  • Prisoner cost: $250/hour
  • Max sentence: 12 hours
  • Bail: $1k standard / $5k mayor / $10k king

Ranks & Permissions

  • co-mayor: All powers except delete town
  • assistant: Invite/kick, claim/unclaim
  • sheriff: Jail management
  • helper: Set permissions

PvP Death Costs

Lose 5% of personal balance (max $1,000) to killer. PvP only — monsters don't cost money.
