Age of Exploration Naval System — Build, Sail, and Conquer

Naval System Overview

Movecraft replaces the standard minecart system with specialized Age of Exploration ship classes. Each ship type has specific building requirements, capabilities, and combat roles. The seas are extremely dangerous — only properly built ships will survive storms, creatures, and enemy fire.

Critical Building Requirements

All ships must meet these material composition requirements:

Planks

30% - 100%

Minimum 30% of your ship MUST be made of planks

Wool (Sails)

4% - 20%

Wool must comprise 4-20% of your total ship volume

Ship Durability

Each ship has two durability thresholds:

  • Sink Percent (80-90%): Ship begins taking damage
  • Overall Sink Percent (60-80%): Ship sinks when this percentage is damaged

Example: Man-o-War sinks at 80% overall damage, making it the most durable vessel

Ship Signs

Required Sign: Every ship MUST have a Craft Type/Name sign. All other signs are optional.

Craft Type Sign REQUIRED

Brigantine

First line must be the craft type (e.g., Pinnace, Cutter, Schooner, Brigantine, Caravel, Corsair, Galleon, Frigate, Man-o-War). This tells Movecraft what class your ship is.

Name Sign

Name:
HMS Nelson

Line 1: "Name:" (exactly as shown). Line 2: Your ship's chosen name. Sets the craft's name displayed on nametag and /contacts.

Helm Sign

[helm]

Allows steering control. Right-click to rotate ship 90° clockwise, left-click to rotate 90° counter-clockwise.

Cruise Sign

Cruise: OFF

Right-click to toggle cruise mode on/off. Cruise mode allows continuous sailing in a straight line.

Pilot Sign

Pilot:
PlayerName

Add registered pilots (one per line). Anyone not on this list cannot pilot the ship. Leave blank for public access.

Manoverboard Sign

Manoverboard

Sets the location that the /manoverboard command will teleport you to. Useful for getting back aboard if you fall off.

Release Sign

Release

Right-click to release the craft (stop sailing). Alternative to /craft release command.

Forbidden Blocks (Tech Restrictions)

The following blocks are prohibited on all Movecraft ships: Pistons, Sticky Pistons, Slime Blocks, Honey Blocks, Observers, Dispensers, Droppers, Redstone components (wire, blocks, repeaters, comparators), Daylight Detectors, Beacons, and Ender Chests. These restrictions maintain Age of Exploration authenticity and prevent exploits.

Ship Progression Path

Start (Civilian): Pinnace (20-150) or Cutter (151-400) for learning and quick travel. Cannons disabled

First Warship: Schooner (500-1400) or Caravel (600-2500) - Your first vessels with cannon capability

Naval Combat: Brigantine (1200-2800) for serious naval engagements

Raiding: Corsair (2500-4400) for hit-and-run tactics (85% durability)

Naval Power: Frigate (4200-6500) for convoy escort and fleet actions (85% durability)

Armed Trade: Galleon (2501-4000) for bulk cargo with cannon protection (85% durability)

End-Game Flagship: Man-o-War (7500-100000) as ultimate naval dominance (90% durability, 80% overall sink)

Movecraft Commands

Basic Controls

/craft sail - Start sailing

/craft release - Stop sailing

/craft info - Ship status

/craft cruise - Toggle cruise mode

/manoverboard - Teleport to manoverboard sign

/contacts - View nearby ships

Combat Commands (Naval Ships Only)

/craft fire - Fire cannons

/craft reload - Reload weapons

/craft repair - Repair damage

/craft pilot - Take control
