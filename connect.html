Internet and Network Troubleshooting

Use a home/personal network

Check your internet connection: Verify that you can visit other websites without issues.

Pause other network traffic: Stop software downloads, video streams, or network-heavy tasks on your gaming device

Check your internet speed: Run an internet provider recommended speed test to check your internet connection speed and latency.

Verify wireless connection strength: If you’re playing on a wireless network, try moving your gaming device closer to your router.

Allow Minecraft through your firewall or VPN: If you have a firewall, antivirus software, or are using a VPN, check the settings to be sure that it is not blocking or slowing your connection.

List of Errors

Connection refused: connect - Confirm you’ve entered the correct server address. If it still doesn’t work, restart your router, flush your DNS, and tweak your firewall, antivirus, or proxy settings. Creeper/ Error Code 221000 - Ensure your internet connection is stable and working properly Crossbow - Close the game completely (ensure it isn’t running in the background either) and let it sit idle for around two minutes before starting it again. End of stream - Restart the application. Haybale - This indicates an Xbox authorization error. First, ensure the Xbox app is up to date. If it is, log out of your account and sign in again. Failed to login: Bad Login - Ensure the version of Minecraft you’re running is the same as the server. If it is, disconnect your wifi, wait a few minutes, and try again. Failed to login: invalid IP - This will either be an internal service error or an error related to your internet. If it’s the former, you’ll have to wait a while; if it’s your internet, you’ll have to troubleshoot it. Failed to login: the authentication servers are currently down for maintenance - Ensure your firewall isn’t causing issues by disabling it temporarily and trying again. If it still doesn’t work and your internet is working fine, then you’ll have to wait for the maintenance to be over. This server responded with an invalid server key - Two things might cause this. Either the server you’re connecting to is outdated, in which case you won’t be able to connect to it, or you will have to log in again. Took too long to log in - Most likely, your internet speed is either too slow or too inconsistent. Failed to check session lock, aborting - Go to the session folder and look for a file that says “.lock’. Delete it and try again. Code – 805306369 - This usually happens because of a lack of memory in your system. Try making more space by deleting stuff or closing some background applications. Code – 1073740940 - Allocate more RAM to the Java Virtual Machine that’s running the game. You’ll have to buy more if you don’t have RAM available. Code – 1073740791 - Update all of your drivers, and this error code should go away. Code: – 1073741819 - This error code is exclusive to Nvidia drivers. You’ll either have to get them updated automatically or manually from the Nvidia website. Game Ended with Bad State: Exit Code 1 - Update JAVA, update all your drivers, and finally, update Minecraft. If the issue remains, uninstall the game and then reinstall it. java.lang.OutOfMemoryError - Allocate more RAM to the Java Virtual Machine that’s running the game. If you don’t have RAM available, then you’ll have to buy more. RuntimeException: No OpenGL context found in the current thread. - Ensure your drivers are up to date. If they are, you’ll likely have to get a better graphics card, as the current one is too weak.
