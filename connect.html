Network Troubleshooting

Use a Personal Network

Connect using a home or personal network. School, work, or public networks often block game servers.

Check Internet Connection

Verify you can visit other websites without issues. Test your connection by browsing or streaming.

Pause Network Traffic

Stop software downloads, video streams, or network-heavy tasks on your gaming device.

Test Internet Speed

Run an internet speed test to check your connection speed and latency. Minimum recommended: 10 Mbps download, 5 Mbps upload.

Check Wireless Strength

If playing on WiFi, move closer to your router or use an ethernet cable for a stable connection.

Firewall & VPN Settings

Allow Minecraft through your firewall, antivirus, or VPN. Temporarily disable them to test connection.

Quick Fixes

Restart Everything

Restart Minecraft, your computer, and your router in that order.

Flush DNS

Open Command Prompt as admin and type: ipconfig /flushdns

Update Java

Ensure you have the latest Java version installed from java.com

Reinstall Minecraft

As a last resort, uninstall and reinstall Minecraft completely.

Common Error Solutions

Click on any error to copy its name to clipboard for further research.

Connection refused: connect

Confirm you've entered the correct server address. If it still doesn't work, restart your router, flush your DNS, and tweak your firewall, antivirus, or proxy settings.

Error Code 221000 (Creeper)

Ensure your internet connection is stable and working properly. Check for ISP outages or network congestion.

Crossbow Error

Close the game completely (ensure it isn't running in the background) and let it sit idle for around two minutes before starting it again.

End of stream

Restart the application. If persistent, check your network connection and try using a different DNS server.

Haybale Error

Xbox authorization error. Ensure the Xbox app is up to date. If it is, log out of your account and sign in again.

Failed to login: Bad Login

Ensure the version of Minecraft you're running is the same as the server. If it is, disconnect your WiFi, wait a few minutes, and try again.

Failed to login: invalid IP

This will either be an internal service error or an error related to your internet. If it's the former, you'll have to wait; if it's your internet, you'll need to troubleshoot it.

Failed to login: authentication servers down

Ensure your firewall isn't causing issues by disabling it temporarily. If it still doesn't work and your internet is fine, wait for Mojang maintenance to complete.

This server responded with an invalid server key

Either the server you're connecting to is outdated (can't connect), or you need to log out and back into your Minecraft account.

Took too long to log in

Most likely, your internet speed is either too slow or too inconsistent. Test your connection and consider upgrading your plan.

Failed to check session lock, aborting

Go to the session folder and look for a file that says ".lock". Delete it and try again. (Location varies by OS)

Code – 805306369

Usually caused by insufficient memory. Free up space by deleting files or closing background applications.

Code – 1073740940

Allocate more RAM to the Java Virtual Machine. In Minecraft launcher settings, increase allocated RAM to at least 4GB.

Code – 1073740791

Update all of your drivers (graphics, chipset, network), and this error code should go away.

Code – 1073741819

Exclusive to Nvidia drivers. Update them automatically via GeForce Experience or manually from the Nvidia website.

Game Ended with Bad State: Exit Code 1

Update Java, update all drivers, and update Minecraft. If the issue remains, uninstall and reinstall the game.

java.lang.OutOfMemoryError

Allocate more RAM to Minecraft. In launcher settings, increase to 4-8GB depending on your system's available memory.

RuntimeException: No OpenGL context found

Ensure your graphics drivers are up to date. If they are, your current graphics card may be too weak for the game's requirements.
